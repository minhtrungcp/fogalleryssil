# gallery App

## Requirements
 * Device android OS > 6.0
 * Clean and rebuild project before run.

## Features:
* Display all image/video in device
* Allow favorite image/video


## Architecture uses in it:
* Clean Architecture
* MVVM

## Tecgnologes uses in it:
 * Kotlin.
 * Navigation components 
 * Hilt
 * Room DataBase
 * Coroutines
 * ExoPlayer
 * Coil
 * Jetpack compose
